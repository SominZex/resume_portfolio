Project Details

The project utilizes publicly available datasets related to COVID-19, including data on confirmed cases, deaths, recoveries, testing rates, vaccination rates, and other relevant metrics. The datasets are sourced from reputable sources such as the World Health Organization (WHO), Centers for Disease Control and Prevention (CDC), and other official health organizations. The project is developed using Python programming language and various libraries and frameworks commonly used in machine learning and data analysis, including but not limited to: NumPy Pandas Scikit-learn TensorFlow Matplotlib Seaborn Project Structure
